# Architecture

## Project Overview
This is a Spring Boot web application leveraging Spring AI.

## Directory Structure Strategy
- `/src/main/java/com/veeral/springai/` - Core Java source code
  - `/controller/` - REST API controllers
  - `/service/` - Business logic and Spring AI client interactions
  - `/model/` - Data records, DTOs, and entities
  - `/config/` - Spring configuration classes
- `/src/main/resources/` - Properties and static assets
- `/.docs/` - System architecture and long-term design documentation
- `/spec.md` - The active feature specification
- `/rules.md` - AI agent coding rules and constraints
